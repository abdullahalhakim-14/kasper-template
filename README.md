# Kasper Template (Landing Page)

Responsive multi-section landing page built with **HTML** and **CSS** (Kasper template).

## Features
- Clean layout with multiple sections (Services / Portfolio / About / Pricing / Contact)
- Responsive design
- Font Awesome icons + Normalize.css
- Video section (webm)

## Tech Stack
- HTML5
- CSS3
- Normalize.css
- Font Awesome
